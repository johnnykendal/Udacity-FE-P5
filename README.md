frontend-nanodegree-neighborhood-map
===============================

### Completed as the fifth project in the Udacity Front-End Nano Degree.

## About
A single page application that displays night clubs in my home city of Kelowna, BC. It uses the the google maps api with the foursquare api to fetch data and display it on google maps. It utilizes the knockout.js, bootstrap.js, bootstrap.css, and jquery.js libraries complied and sorted using bower.

### Loading the site

1. Download the repository

2. Run a local server in terminal

  ```bash
  $> cd /path/to/your-project-folder
  $> python -m SimpleHTTPServer 8080
  ```

3. Open a browser and visit localhost:8080
